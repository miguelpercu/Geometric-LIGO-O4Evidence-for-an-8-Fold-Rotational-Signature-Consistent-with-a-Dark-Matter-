# Geometric-LIGO-O4Evidence-for-an-8-Fold-Rotational-Signature-Consistent-with-a-Dark-Matter-
This deposit contains the dataset, complete analytical pipeline, and preprint manuscript reporting the detection of a statistically significant geometric phase quantization in persistent spectral lines common to the LIGO Hanford (H1) and Livingston (L1) detectors during the O4a run.
